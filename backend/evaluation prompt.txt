# ✂️ TASK 2: SAM2 SEGMENTATION EVALUATION

We do NOT have ground-truth masks → design proxy metrics carefully.

### Metrics:

1. **Mask-to-box IoU**

   * Compare SAM mask vs YOLO bounding box

2. **Mask coverage ratio**

   * mask_area / bbox_area
   * Detect overfill (≈1.0 → bad)

3. **Compactness score**

   * area / convex hull area

4. **Edge leakage**

   * % of mask pixels outside bbox

5. **Multi-mask stability**

   * Check if different prompts give similar masks

### Detect failure cases:

* Full rectangle masks
* Extremely small masks
* Broken/disconnected masks

### Output:

* IoU histogram
* mask quality score (0–1)
* worst 20 masks saved as images
